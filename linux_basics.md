# Linux Basics Notes

## Basic Commands

pwd - Shows current directory

ls - Lists files and folders

cd - Changes directory

mkdir - Creates a new folder

rm - Removes a file

cat - Displays file contents

touch - Creates a new file

cp - Copies files

mv - Moves or renames files

clear - Clears terminal screen

## Why Linux is Important in Cybersecurity

Linux is widely used in cybersecurity because many servers, cloud systems, and security tools run on Linux. Learning Linux helps in understanding system administration and security testing.
