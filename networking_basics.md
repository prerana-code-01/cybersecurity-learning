# Networking Basics Notes

## IP Address
A unique address assigned to a device on a network.

## DNS
Domain Name System converts website names into IP addresses.

## HTTP
Protocol used for communication between web browsers and servers.

## HTTPS
Secure version of HTTP using encryption.

## TCP
Reliable communication protocol.

## UDP
Faster but less reliable communication protocol.

## Why Networking Matters in Cybersecurity

Cybersecurity professionals must understand how devices communicate to identify and prevent attacks.
